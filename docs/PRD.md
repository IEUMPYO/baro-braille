# PRD: 바로점자

## 목표

수능 수학 문제를 점자로 변환하는 웹 데모. 교육청 담당자들에게 보여줄 프로토타입.

## 사용자

- 교육청 점역 담당자
- 특수교육 지원센터
- 최종적으로는 시각장애인 학생

## 핵심 기능

1. 파일 업로드 (PDF, JPG, PNG)
2. 수식/그래프 인식 및 미리보기
3. 점역 변환
4. BRF/전자점자/PDF 다운로드

## 현재 상태

Mock data로 플로우만 구현. 백엔드/AI 없음. 기관 데모용.

## 안 만드는 것

- 실제 OCR/AI 연동
- 백엔드 서버
- 로그인/권한
- DB
- 사용량 추적
- 배치 처리
- 히스토리

## 나중에 할 것

- Mock → API 전환
- AI 모델 붙이기
- GitHub Pages → 클라우드 서버
- 계정/쿼터 관리

## 디자인

밝은 톤. 블루 계열. 카드 레이아웃. 그라데이션 배경.
수학 문제는 세리프체, UI는 산세리프.
